* Build a real-time fraud detection pipeline using PySpark and Spark Streaming on distributed data. 
* Engineered features and trained classification models with Spark MLlib on anonymized transaction data. 
* Deployed pipeline components on AWS for scalability; achieved 99.99% accuracy and 0.999 AUC-ROC score.
